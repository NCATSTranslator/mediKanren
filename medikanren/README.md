# MediKanren

## Directory Structure

| Directory | Description |
|---|---|
|[apps](apps)| Web service and local GUI applications |
|[util/data-import](util/data-import)| Import tools from external KG formats |
|[open-api](open-api)| Module for integration with external KGs |
|[pieces-parts](pieces-parts)| Extensions of core API |
|[data](data)| Install KG data here (.gitignored) |
|[test](test)| Collection of test code |
|[test/unit](test/unit)| Tests without external dependencies |
|[test/full-dbs](test/full-dbs)| Tests depending on full KG |
|[test/external-http](test/external-http)| Tests depending on external http services |
|[use-cases](use-cases)| Moved to [contrib/medikanren/use-cases](../contrib/medikanren/use-cases) |
